## 🧠 Skills Covered

**Active Directory & User Management**
- Provisioned and managed domain using Samba4 AD
- Created, disabled, and enabled user accounts in a domain environment

**Email & Communication Systems**
- Diagnosed SMTP, IMAP, and POP3 mail delivery issues
- Resolved DNS MX record misconfigurations affecting email routing
- Tested mail ports for connectivity and firewall blocking

**System Administration**
- Resolved Windows Update errors and deployed patches using WSUS
- Diagnosed driver conflicts, OS errors, and system performance issues using Event Viewer and Task Manager
- Troubleshot Remote Desktop Protocol (RDP) errors and VPN connectivity failures

**Network Troubleshooting**
- Resolved APIPA addressing, duplicate IP conflicts, and default gateway misconfigurations
- Diagnosed LAN and Wi-Fi issues using `ipconfig`, `arp`, and wireless diagnostics
- Resolved DNS resolution failures using `nslookup`, `dig`, and DNS cache flush

**Print Services**
- Fixed print spooler failures and driver conflicts
- Configured and troubleshot network printers across subnets
